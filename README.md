## 🧠 주요 프로젝트

### 📌 Vehicle License Plate De-identification (CV)
- **기술**: YOLO, ResNet, OpenCV, AdaIN, ...
- **내용**:
  - 차량 번호판 탐지 및 비식별화
  - 합성 데이터 생성을 위한 이미지 파이프라인 구축
  - 실제 데이터 활용을 고려한 전처리 및 안정성 중심 설계

👉 Repo: https://github.com/seung23/license-plate-deid

---

### 📌 PAZULE (Vision 기반 프로젝트)
- **기술**: BLIP, CLIP, ...
- **내용**:
  - 이미지 인식 기반 문제 해결 프로젝트
  - Vision task에서의 데이터 처리 경험

👉 Repo: https://github.com/seung23/PAZULE

---

### 📌 Movie Recommendation System (RecSys)
- **기술**: EASE, LightGCN, MultiVAE, BERT4Rec, ...
- **내용**:
  - 사용자–아이템 상호작용 기반 추천
  - Side Information을 활용한 Hybrid 접근
  - Recall@10 중심의 실험 설계

👉 Repo: https://github.com/seung23/pro-recsys-movierecommendation-recsys-04
